# Invoice Tracker (Next.js Pages Router)

A minimal invoice management app built with Next.js Pages Router.

## Setup

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Features

- Dashboard with invoice summary statistics
- Invoice management (list, create, update, delete)
- Customer directory
- Mock API routes with in-memory data
- Simulated authentication (hardcoded user)
- Dark theme UI with sidebar navigation
